# browser

![Actions status](https://github.com/KevinL10/browser/actions/workflows/CI.yml/badge.svg)


A web browser implemented from scratch in Python.


Adapted from [browser.engineering](https://browser.engineering/).



### Supported features:

ch. 1
- `HTTP/1.1`
- File URI scheme
- data scheme
- view-source scheme
